# PasteBin

一个基于MongoDB的Pastebin服务

示例：[https://pastebin.guoziyang.top](https://pastebin.guoziyang.top)

采用了Spring Boot + MongoDB，前后端不分离，模版引擎使用的Thymeleaf

特性：
- 多语言高亮支持，Markdown渲染
- 支持密码访问
- 定期删除，自定义删除时间
- 支持阅后即焚
